Slovníček pojmů:

  • Proprietární software -> Je software, kde jeho autor upravuje licencí či jiným způsobem možnosti jeho používání. K takovému software nejsou k dispozici volně zdrojové kódy či v nich nelze svobodně dělat úpravy a výsledné dílo distribuovat. Takový software obvykle spadá do kategorie komerčního software, který jeho autor prodává.
  • FSB -> Je sběrnice systému jenž propojuje procesor a operační paměť RAM.
  • RAID -> Vícenásobné diskové pole nezávislých disků. Typ diskových řadičů, které zabezpečují pomocí určitých speciálních funkcí koordinovanou práci dvou nebo více fyzických diskových jednotek.
  • HyperTransport -> Je přenosová rychlost (FSB) 2000MT/s (1GHz).
  • Technologie MuTIOL 1GVGA -> Sběrnice která propojuje Southbridge a Northbridge, přenosová rychlost 1GB/s (patent společnosti SiS).
  • FlexMemory Technology -> Je možnost osazení základní desky paměťovými moduly o různých kapacitách a přesto využívat dvoukanálový režim (u dřívější generace musely být paměťové moduly pro zachování dvoukanálového režimu o stejné kapacitě).
  • Turbo Memory (Robson) -> Technologie zkracuje čekací doby u často používaných aplikací až o polovinu. Je to flash paměť, na kterou se ukládají soubory, ke kterým je třeba častější a hlavně rychlejší přístup, neboť flash paměť NAND je rychlejší než pevný disk.
  • Quick Resume Technology -> Vlastnost rychlého zapnutí a vypnutí čipsetu (po vypnutí je v režimu spánku).
  • Quiet System Technology -> Jižní můstek nabízí čtyři senzory pro měření otáček ventilátorů a tři výstupy PWM pro jejich řízení. Nastavení je přístupné v BIOSu.
  • Matrix Storage Technology -> Je podpora RAID 0, 1, 5 a 10.
  • PCI -> Je počítačová sběrnice pro připojení periferií k základní desce, která není omezená na platformu osobních počítačů PC. Používá paralelní přenos dat.
  • SATA -> Serial ATA je počítačová sběrnice využívaná pro připojení dalších zařízení (pevné disky, optické disky). Disky se připojují přímým a samostatným kabelem k řadiči a tím pádem se nemusí rozlišovat na Master, Slave a Cable Select.
  • DirectX -> Rozhraní mezi hardwarem a softwarem. Konkrétně mezi Windows a grafickými kartami, zvukovkami, myšmi, klávesnicemi, atd. Jejich cílem je usnadnit programátorovi přístup k jejich funkcím.
